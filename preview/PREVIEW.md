# 👀 Локальний перегляд сайту (без налаштувань)

Це швидкий спосіб побачити, **як виглядає сайт**, не підключаючи Supabase, Cloudinary чи бота.
Він використовує **вигадані товари** (8 штук) і картинки-заглушки.

## Як запустити

Потрібен лише **Node.js** (ніяких `npm install` — превью без залежностей).

```bash
node preview/preview-server.mjs
```

Потім відкрийте у браузері:

```
http://localhost:8080
```

Щоб змінити порт: `PORT=3001 node preview/preview-server.mjs`

## Що можна подивитися

- Головна, каталог, категорії, картка товару (галерея, фасування, відгуки)
- Кошик (додавання/зміна кількості працюють)
- Оформлення замовлення, сторінка «Дякуємо»
- Блог і статті, сторінки Про нас / Доставка / Контакти
- `/sitemap.xml` і `/robots.txt`

## Вхід через Telegram у превью

Справжня кнопка Telegram локально не працює (Telegram вимагає реальний домен у BotFather).
Щоб побачити стан «увійшов»:

- **Демо-вхід:** відкрийте `http://localhost:8080/auth/demo`
- **Вихід:** `http://localhost:8080/auth/logout`

## Важливо

Це **лише візуальний превью**. Реальні товари, фото/відео та збереження замовлень
працюють лише у повному режимі (`npm start` із реальною БД — див. `WEBSITE.md`).
Картинки у превью беруться з picsum.photos, тому потрібен інтернет.
