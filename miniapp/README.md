# WowSmak — Telegram Mini App

Статический фронтенд (HTML + CSS + JS, без сборщика). Раздаётся тем же Express-сервером по пути `/app/`.

## Как работает
- Открывается из бота: кнопка меню чата «🍬 Магазин» или inline-кнопка в `/start`.
- Данные берёт через `/api/*` на том же сервере. Каждый запрос проверяется по подписи Telegram `initData` (HMAC-SHA256 с `BOT_TOKEN`) — подделать нельзя.
- Supabase service_role ключ остаётся только на сервере — фронтенд к БД напрямую не ходит.

## Экраны
- Главная (хиты/новинки, категории)
- Каталог с фильтром по категориям
- Карточка товара (фото, видео, акция, остаток, описание)
- Корзина (± количество, очистка, итоги)
- Оформление (имя, телефон, адрес, комментарий) + экран успеха
- Мои заказы (статусы)
- Профиль (имя, статистика)

## Локальная разработка без Telegram
В режиме без `PUBLIC_URL` можно задать `DEV_TG_ID=<ваш tg id>` в `.env` — тогда API работает без подписи (только для локального теста, открывайте http://localhost:3000/app/).
В продакшене (с `PUBLIC_URL`) dev-режим отключён.

## Структура
```
miniapp/
  index.html        — оболочка + подключение Telegram SDK
  css/styles.css    — тёмная неоновая тема
  js/telegram.js    — обёртка над WebApp SDK
  js/api.js         — fetch с заголовком initData
  js/store.js       — глобальное состояние
  js/app.js         — роутер и экраны
```
