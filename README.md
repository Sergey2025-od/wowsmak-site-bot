# 🍬 Candy Bot — Telegram-магазин конфет

Свой бот без конструкторов, на стеке:

- **grammY** — фреймворк для Telegram-ботов (Node.js)
- **Supabase** — база данных (PostgreSQL)
- **Cloudinary** — хранение и оптимизация картинок/видео
- **GitHub + Render** — код и хостинг (webhook)

## Функции

- 🍬 Каталог по категориям с фото и ценами
- 🛒 Корзина (− / +, очистка)
- ✅ Оформление заказа (имя, телефон, адрес, комментарий)
- 🔔 Уведомление админу о новом заказе
- 📦 «Мои заказы» с историей и статусами
- Оплата пока не подключена (заказ подтверждает менеджер)

## Быстрый старт

### 1. Бот в Telegram
Откройте [@BotFather](https://t.me/BotFather) → `/newbot` → получите `BOT_TOKEN`.

### 2. Supabase
1. Создайте проект на [supabase.com](https://supabase.com).
2. SQL Editor → выполните `supabase/schema.sql`.
3. Settings → API → скопируйте `Project URL` и `service_role` ключ.

### 3. Cloudinary
Зарегистрируйтесь на [cloudinary.com](https://cloudinary.com), возьмите `cloud_name`, `api_key`, `api_secret`.
В `products.image_url` храните `public_id` картинки (или полный https-URL).

### 4. Локальный запуск
```bash
cp .env.example .env   # заполните ключи
npm install
npm run seed           # (опционально) примеры товаров
npm run dev            # без PUBLIC_URL работает через polling
```

### 5. Деплой на Render
1. Залейте код на GitHub.
2. На [render.com](https://render.com) → New → Blueprint → выберите репо (подхватит `render.yaml`).
3. Впишите переменные окружения (из `.env.example`).
4. `PUBLIC_URL` — адрес вашего сервиса на Render (напр. `https://candy-bot.onrender.com`).
5. После деплоя бот сам установит webhook.

> ⚠️ На бесплатном плане Render сервис «засыпает» при простое. Первый запрос после сна может идти ~30 сек. Для стабильности — платный план или cron-ping.

## 🔐 Админ-панель

Доступ только у тех, чьи Telegram id указаны в `ADMIN_CHAT_ID` (можно несколько через запятую).

### Подключение
1. Выполните миграцию `supabase/admin-migration.sql` в Supabase → SQL Editor (добавляет `stock`, `sale_price`, `video_url` и таблицу `messages`).
2. Убедитесь, что `ADMIN_CHAT_ID` содержит ваш id (узнать можно у @userinfobot).
3. Отправьте боту `/admin`.

### Возможности
- **➕ Добавить товар** — пошагово: категория → название → описание → цена → остаток → фото.
- **📝 Товары** — редактирование любого поля (название, описание, цена, остаток, акция, фото, видео) и удаление.
- **🏷 Акции** — назначение/снятие акционной цены (в витрине старая цена зачёркнута).
- **Остатки** — товар с нулевым остатком скрывается из витрины; при заказе остаток автоматически списывается. `null` = не учитывать.
- **📢 Рассылка** — сообщение (текст или фото+подпись) всем клиентам от имени магазина.
- **Связь с клиентом** — любое сообщение клиента (не кнопка меню) пересылается админам с кнопкой «✍️ Ответить». Ответ уходит клиенту от имени магазина.
- **Фото/видео** — загружаются в Cloudinary прямо из Telegram (отправьте файл боту в нужный момент).

### Команды
- `/admin` — открыть панель
- `/cancel` — отменить текущее действие
- `/skip` — пропустить загрузку фото/видео
